1.You need to have a text file in your directory called Manifest.txt and it has to have Main-Class: Frame in it followed by a new line Which should be there already.
2.Then you need to open the terminal and cd to directory, and type in: jar cfm AccountManager.jar Manifest.txt *.class
3.After that you can just click on the jar file to run, or type java -jar AccountManager.jar in terminal and it will run.
